# keke-birthday

可可 20 岁生日 H5。

GitHub Pages 理论访问地址：

https://lbql777.github.io/keke-birthday/

## Files

- `index.html`：正式入口页面，原生 HTML/CSS/JavaScript 单文件。
- `photo1.jpg`、`photo2.jpg`、`photo3.jpg`：生日照片资源。
- `music.mp3`：原创合成生日氛围 BGM。页面会在用户点击后播放；如果文件缺失或浏览器拒绝播放，会显示友好提示，不影响页面继续使用。
